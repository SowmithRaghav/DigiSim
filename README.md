# DigiSim: Digital Logic & Signal Transition Visualizer

An ECE-focused software simulator written in C++ to model behavioral logic gates, propagation delays, and timing hazards in combinational circuits.

## Tech Stack & Tools
- Language: C++17
- Analysis: GDB Trace Registers
- Framework: Standard File-Stream Netlist Parser
